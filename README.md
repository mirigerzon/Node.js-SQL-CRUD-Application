# Node.js SQL CRUD Application

A RESTful API application built with **Node.js**, **Express.js**, and **MySQL** for managing data using CRUD operations (Create, Read, Update, Delete).  
This project demonstrates backend development skills, database integration, and API design.

---

## Features
- CRUD operations for managing data in a SQL database  
- RESTful API endpoints built with Express.js  
- Data validation and error handling  
- Optional JWT authentication for secure endpoints  

---

## Technologies
- Node.js  
- Express.js  
- MySQL  
- Sequelize ORM (optional)  
- JWT Authentication (optional)  

---

## Purpose
This project is designed to showcase backend development skills using Node.js and SQL.  
It provides a simple yet effective example of building a RESTful API with a relational database, demonstrating data management and server-side programming concepts.
